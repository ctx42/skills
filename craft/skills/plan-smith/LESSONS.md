# Lessons

Rules learned for the `plan-smith` skill. Read before running; obey each line.

- Keep summary-table Item cells short (≤ ~30 chars) and pad every cell to its
  column's widest cell so the `|` delimiters align — a long name in one row
  otherwise skews the whole table.
- Write the separator row with dashes flush to the pipes, no surrounding spaces
  (`|----|`, not `| -- |`), one dash per character of column width; keep columns
  tight to the widest cell, not over-padded.
